# PAC BATTLE ARENA

TikTok Live Pac-Man Arena Game - Multiplayer battle game with Blue vs Pink teams, TikTok gift/like/chat integration, and real-time action.

## Features

- 20 AI Pac-Mans always active (game never feels empty)
- Blue vs Pink team system (join via TikTok gifts)
- Activity-based combat (active beats inactive)
- TikTok Live integration (gifts, likes, chat commands)
- Gift power-ups: Boxing Glove, Donut, Shell, Money Magnet
- Chat commands: BLUE, PINK, BOOST, ATTACK, CHAOS
- 15-minute rounds with Top 3 + Team Winner
- King crown for top player
- Win tracking across rounds
- 1:1 square format, OBS browser source ready

## Tech Stack

- **Frontend:** HTML5 Canvas (PixiJS)
- **Backend:** Node.js + Express + WebSocket
- **TikTok:** TikTok-Live-Connector

## Setup

```bash
npm install
npm start
```

Open `http://localhost:3000` in OBS as browser source (1080x1080).
